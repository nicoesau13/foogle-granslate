## How to Use

1. Open `index.html` in your web browser
2. Select your source language (or use "Detect Language")
3. Select your target language
4. Type or paste text in the left text box
5. The translation will appear automatically in the right text box
6. Use the swap button (‚áÑ) to switch languages
7. Click "Copy" to copy the translation to your clipboard
8. Click "Clear" to reset the text

## Installation

No installation required! Simply open `index.html` in any modern web browser.

```bash
# Clone the repository
git clone https://github.com/nicoesau13/foogle-granslate.git

# Navigate to the directory
cd foogle-granslate

# Open index.html in your browser
# On macOS:
open index.html
# On Linux:
xdg-open index.html
# On Windows:
start index.html
```

## Technical Details

This is a client-side web application built with:
- HTML5
- CSS3 (with Flexbox and Grid)
- Vanilla JavaScript

**Note:** This version uses a mock translation database for demonstration purposes. In a production environment, you would integrate with a real translation API such as:
- Google Translate API
- Microsoft Translator API
- LibreTranslate (open-source)
- DeepL API

## Development

To extend this project:

1. Add more languages to the dropdown menus in `index.html`
2. Add corresponding translations to the `translations` object in `script.js`
3. Integrate a real translation API for production use
4. Customize the styling in `style.css`
